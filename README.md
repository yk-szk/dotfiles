# dotfiles

## INSTALLATION

```sh
git clone https://github.com/yk-szk/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
bash ./install.sh
```

### bash (optional)

```sh
cd ~/.dotfiles
bash ./init-bash.sh
```

### In Windows
In admin cmd

```bat
cd %HOMEPATH%
git clone https://github.com/yk-szk/dotfiles.git
cd dotfiles
install.bat
```
